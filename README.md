# W3DS TRL7 Documentation

This repository contains the documentation for the **W3DS TRL7** project.

W3DS (Web 3 Data Space) is a post-platform infrastructure approach where:
- data is owned and controlled by users (eVaults)
- services operate on user-controlled data instead of storing it
- identity is sovereign and reusable across services
- security is built into the infrastructure layer

This repository focuses on:  
Infrastructure layer up to platform adapters (inclusive)

---

# Principles

- Documentation is the primary product at this stage  
- We work iteratively (Agile), not trying to make everything perfect from the start  
- We learn and improve the process continuously  
- GitHub is the single source of truth  
- Decisions must be documented
- We can collectively agree to change the documentation development process if a more effective one is proposed

---

# Scope

**Included:**
- Core infrastructure components (eVault, Registry, etc.)  
- Identity and authentication model  
- Data access and interaction patterns  
- Platform adapters  

**Not included:**
- UI/UX   
- End-user applications  
- Implementation details beyond architecture/protocol level
